<h1 align="center">Clinic Web App </h1>

<p align="center">
A responsive landing page designed to promote a clinic management system.
</p>

<hr>

<h2>📌 Overview</h2>

<p>
This project is a modern landing page built to showcase the features of a clinic
web application. The page is designed to attract doctors and clinic owners by
highlighting key benefits such as appointment booking, clinic management,
and easy communication with patients.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>Responsive design (mobile, tablet, desktop)</li>
<li>Modern UI layout</li>
<li>Service and feature sections</li>
<li>Call-to-action for clinic owners</li>
<li>WhatsApp contact integration</li>
<li>Google Maps clinic location integration</li>
</ul>
